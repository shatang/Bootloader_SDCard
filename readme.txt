/********************************************************************************
* 函数名 : IAP_SDCard
* 功  能 : IAP功能，使用SD卡进行升级
* 说  明 : 基于Stm32F407
* 入  参 : none
* 返  回 : none
* 设  计 : Shatang                    时  间 : 2020.09.17
* 修  改 : none                       时  间 : none
********************************************************************************/
1. 该IAP升级，通过读取SD卡的文件进行APP升级
2. 文件名、类型：Shatang.bin；路径是SD卡的家目录，且不支持中文文件名(删减过FatFs库的字库)
3. 该Bootloader进行升级需要重新上电启动
4. 上电2s检测是否插入SD卡，2s后检测不到(或升级完毕)进入App程序
5. 无升级则LED指示灯一直亮，直到跳转至APP；进行升级时灭灯
6. Bootloader程序总大小为20KB
